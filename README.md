# 📱 Jobs Assignment App

A simple React Native (Expo + TypeScript) application with a minimal Node.js backend that allows service providers to view and manage their daily jobs.

---

##  Tech Stack

### Mobile
- React Native (Expo)
- TypeScript
- React Navigation

##  Features

-  Jobs List (title, time, area, status)
-  Job Detail screen
-  Mark job as completed (with validation)
-  Auto refresh after completion
-  In-memory backend (mock data)
